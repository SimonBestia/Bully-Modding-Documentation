SoundFX IDs

Notes:

The list is all the SoundFX IDs from the Debug Wii Build's .act and .lua files. As such, some might not work if you're trying to use them on a different platform.
(Keep in mind this is likely not all of them)

Note that they are stored among various .SEA files for PC, or .BNK for PS2.
They need to be loaded first before you can play them.

Foobar2000 supports .SEA files out of the box, so it's easy to tell which bank SFX are stored in.
