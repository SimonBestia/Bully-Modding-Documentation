Actual limitations

Due to having little RAM to use, things that are very common on mods made for PC cannot be done on the PS2:
  • Loading every Animation Group of the game to avoid A-Posing.
    Rockstar Vancouver devs copy-pasted the same Animation in MANY different Animation Groups, creating very specific set of Animations to use for specific missions to avoid loading any unnecessary ones. Loading more than 16 leads to memory corruption and most likely a crash. While this is a problem for all platforms and should be avoided, the strict limit of available memory here cannot do much, so try not to load that many groups.
  • Keeping multiple Threads running at the same time.
