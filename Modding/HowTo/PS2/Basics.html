Basics: PS2

First of all, getting in the files requires you to extract the contents of the .ISO.
Windows will let you double-click on it so you can access them all pretty easily, but if you edit any you'll need to repack them.

You'll need UltraISO to be able to move modified files directly into an .ISO, but editing a large one like BULLY's requires a paid key. You know what to do.
I have an old video showing how to use the program, but it should be straightforward.
Make sure to untick "Backup Files on Save (.uibak)" in the Save settings for faster saving. Make your own backup of the game's .ISO beforehand!

Edit files like you normally would on PC, but keep Differences in mind.

Tools:

Recommended .IMG Editor: IMG Factory

Blender Plugin for .DFF models: DragonFF
