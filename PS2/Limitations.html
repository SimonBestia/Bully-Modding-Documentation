Actual limitations

Due to having little RAM to use, things that are very common on mods made for PC cannot be done on the PS2:
  • Loading every Animation Group of the game to avoid A-Posing. (Forget about mods like Super Mod running properly on the PS2)
  • Keeping multiple Threads running at the same time.
    • Rockstar Vancouver did things like copy-pasting the same Animation in MANY different Animation Groups, leading to a very specific set of Animations to use for specific missions, and avoid loading unnecessary ones.
    Check out the Common Misconceptions page for more information.
